## 1. `실행취소 이력` 대화상자 → `실행취소`
`실행취소(Undo)` 버튼을 누르면, `실행취소 이력` 대화상자의 목록의 한 단계 전의 상태로 돌아갑니다.

<a id="15-02-07-02-01-s2"></a>

## 2. `이미지 메뉴 바` → `편집` → `실행 취소`
`이미지 메뉴 바` → `편집` → `실행 취소` 메뉴를 클릭하면, `실행취소(Undo)` 버튼을 누른 것과 같은 동작을 합니다.

## 3. Ctrl(⌘) + `Z`
Ctrl(⌘) + `Z` 키를 누르면, `실행취소(Undo)` 버튼을 누른 것과 같은 동작을 합니다.

<a id="90-10-46-01-01-a1"></a>

#### [영상 90.10.46.1.1.a1. Ctrl(⌘) + `Z`: 실행취소(Undo) (Windows) (우리말)](./90-10-46-01-01-undo.md#90-10-46-01-01-a1)

<a id="15-02-07-02-01-s4"></a>

## 4. `위쪽 방향키(↑, Up Arrow)`로 `실행 취소`하기

위쪽 방향키를 눌러서 [실행 취소(Undo)](./16-03-02-00-undo.md)를 할 수 있습니다.

이 기능은 특히 복잡한 작업에서 여러 단계를 실행 취소를 해야 할 때 유용합니다.

[Ctrl(⌘) + `Z`](./15-02-07-02-01-undo.md#15-02-07-02-01-s3)를 여러번 누르는 것보다 더 쉽습니다.

<a id="90-04-10-a111"></a>

#### [영상 90.4.10.a111. `실행취소 이력` 대화상자: `위쪽 방향키(↑, Up Arrow)`로 `실행 취소`하기 (Windows) (우리말)](./90-04-0010-undo_history.md#90-04-10-a111)
