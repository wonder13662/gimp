# 15.3.2.2.4.2. 붓 편집(Edit Brush)

<a id="90-04-15-a101-01-02"></a>

#### [그림 90.4.15.a101.1.2. `붓` 대화상자 → 아래쪽 버튼들과 슬라이더 → `붓 편집` (Windows) (우리말)](./90-04-0015-brushes.md#90-04-15-a101-01-02)
<img width="200" height="273" alt="90-04-15-a101-01-02" src="https://github.com/user-attachments/assets/4c412b59-68eb-46cd-b28d-6528431b56d3" />

`붓` 대화상자 → 아래쪽 버튼들과 슬라이더 → `붓 편집` 버튼을 누르면 활성화된 붓의 [붓 편집기](./15-03-02-03-00-brush_editor.md) 대화상자가 열립니다.

[붓 편집기](./15-03-02-03-00-brush_editor.md) 대화상자는 [7.6.5. 파라매트릭 붓(parametric brush)](./07-06-05-paramatric_brush.md)에 대해서만 작동합니다.

<a id="90-04-15-a101-01-02-01"></a>

#### [영상 90.4.15.a101.1.2.1. `붓` 대화상자 → 아래쪽 버튼들과 슬라이더 → `붓 편집` (Windows) (우리말)](./90-04-0015-brushes.md#90-04-15-a101-01-02-01)
<video controls="controls" width="640" height="360" src="https://github.com/user-attachments/assets/821d9be1-1253-498d-8cbc-df3b6a28ab02"></video>

다른 타입의 붓에 대해서는 [붓 편집기](./15-03-02-03-00-brush_editor.md) 대화상자에서 붓 정보를 표시해주지만, 아무것도 수정할 수 없습니다.
