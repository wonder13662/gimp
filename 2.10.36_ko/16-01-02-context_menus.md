# 16.1.2. 컨텍스트 메뉴(Context Menu)
GIMP(김프) 인터페이스에서 오른쪽 클릭을 하면, `컨텍스트 메뉴(Context Menu)`가 열립니다.

`컨텍스트 메뉴(Context Menu)`에서 다양한 기능을 사용할 수 있습니다.

`컨텍스트 메뉴(Context Menu)`를 열 수 있는 위치는 아래와 같습니다.

<a id="16-01-02-s1"></a>

`이미지 메뉴(Image Menu)`는 [이미지 메뉴 바](./19-glossaryx-image_menu_bar.md)와 동일한 메뉴 항목을 가지고 있습니다.

## 3. 이미지 메뉴 바에서의 오른쪽 클릭
[이미지 메뉴 바](./19-glossaryx-image_menu_bar.md)에서 오른쪽 클릭을 하면, 왼쪽 클릭과 같은 동작을 합니다.

<a id="16-01-02-s4"></a>

## 4. 타이틀 바의 컨텍스트 메뉴
[타이틀 바](./19-glossaryx-title_bar.md)에서 오른쪽 클릭을 하게되면, GIMP(김프)에 속하지 않은 컨텍스트 메뉴가 열립니다.

이 컨텍스트 메뉴는 사용자의 운영체제의 창 관리자 프로그램의 메뉴입니다.
