# 15.5.1.2.1.4. 이 도구 프리셋 되돌리기(Restore this tool preset)

<a id="90-04-22-a154"></a>

#### [그림 90.4.22.a154. 도구 프리셋 대화상자 → 아래쪽 버튼들 → `이 도구 프리셋 되돌리기` (Windows) (우리말)](./90-04-0022-tool_presets.md#90-04-22-a154)

`이 도구 프리셋 되돌리기(Restore this tool preset)` 버튼을 누르면, 선택한 도구 프리셋에 저장된 도구 옵션의 설정값으로 되돌립니다.
