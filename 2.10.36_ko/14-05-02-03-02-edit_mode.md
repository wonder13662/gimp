# 14.5.2.3.2. 편집 모드 - 편집(Edit Mode)

<a id="90-03-02-35-a101-01-02"></a>

#### [그림 90.3.2.35.a101.1.2. `도구 옵션` → `경로` → `편집 모드` → `편집` (Windows) (우리말)](./90-03-02-35-paths.md#90-03-02-35-a101-01-02)
![90-03-02-35-a101-01-02](https://github.com/wonder13662/gimp/assets/15767104/993e6a90-6dda-4b73-baec-96e300253e86)

`편집`에서는 만들어진 경로에 대해서만 작업할 수 있습니다. 

<a id="14-05-02-03-02-s1"></a>

## 1. 마우스 포인터: 편집 불가

마우스 포인터가 경로가 아닌 곳에 있으면, 마우스 포인터의 아이콘이 선이 그어진 원으로 바뀝니다. 이 아이콘은 경로를 수정할 수 없다는 의미입니다.

<a id="90-03-02-35-a201-03"></a>

#### [그림 90.3.2.35.a201.3. 마우스 포인터 아이콘(편집 불가)](./90-03-02-35-paths.md#90-03-02-35-a201-03)
![90-03-02-35-a201-03](https://github.com/wonder13662/gimp/assets/15767104/2a74eb3c-d47e-4c27-815b-2f0b07611812)

<a id="14-05-02-03-02-s2"></a>

## 2. 두 개의 제어점 사이에 선분 추가하기(Add a segment between two nodes)
한 쪽의 [제어점](./19-glossaryx-path_control_point.md)을 클릭해서 활성화시킵니다. 마우스 포인터가 합집합을 나타내는 고리모양으로 바뀝니다. 

<a id="90-03-02-35-a201-04"></a>

#### [그림 90.3.2.35.a201.4. 마우스 포인터 아이콘(합집합 고리모양)](./90-03-02-35-paths.md#90-03-02-35-a201-04)
![90-03-02-35-a201-04](https://github.com/wonder13662/gimp/assets/15767104/1ec5b610-a5f6-4909-8514-b86937af3976)

다른 쪽의 제어점을 클릭해서 두 제어점을 연결합니다. 이 기능은 열려있는 [컴포넌트](./19-glossaryx-path_component.md)를 닫을 때도 유용합니다.

<a id="90-03-02-35-a101-01-02-01"></a>

#### [영상 90.3.2.35.a101.1.2.1. `도구 옵션` → `경로` → `편집 모드` → `편집`: 두 개의 제어점 사이에 선분 추가하기 (Windows) (우리말)](./90-03-02-35-paths.md#90-03-02-35-a101-01-02-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/cde5ac0e-9eaa-421c-a189-c8e23edfc1a0"></video>

<a id="14-05-02-03-02-s3"></a>

## 3. 경로에서 선분을 지우기(Remove a segment from a path)
Ctrl(⌘) + Shift(⇧) 키를 누른 상태에서 마우스 포인터를 [선분](./19-glossaryx-path_segment.md) 위로 가져가면, 마우스 포인터 아이콘이 `-` 기호로 바뀝니다.

<a id="90-03-02-35-a201-05"></a>

#### [그림 90.3.2.35.a201.5. 마우스 포인터 아이콘(`-`)](./90-03-02-35-paths.md#90-03-02-35-a201-05)
![90-03-02-35-a201-05](https://github.com/wonder13662/gimp/assets/15767104/49bb494e-5981-4423-9922-82ff580966ff)

클릭을 하면, 선분이 지워집니다.

<a id="90-10-35-02-03-a1"></a>

#### [영상 90.10.35.2.3.a1. Ctrl(⌘) + Shift(⇧): 선분 지우기](./90-10-35-02-03-remove_segment.md#90-10-35-02-03-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/ce1ca37e-0556-45c2-bea2-a4f7cbb6f4e5"></video>

<a id="14-05-02-03-02-s4"></a>

## 4. 경로에 노드 추가하기(Add a node to a path)
[선분](./19-glossaryx-path_segment.md)을 클릭합니다. 마우스 포인터가 `+` 기호로 바뀝니다.

<a id="90-03-02-35-a201-01"></a>

#### [그림 90.3.2.35.a201.1. 마우스 포인터 아이콘(`+`)](./90-03-02-35-paths.md#90-03-02-35-a201-01)
![90-03-02-35-a201-01](https://github.com/wonder13662/gimp/assets/15767104/7587a0f4-db0c-4443-80a7-d8d3d727384c)

[제어점](./19-glossaryx-path_control_point.md)을 추가하고 싶은 곳에 클릭합니다.

<a id="90-03-02-35-a101-01-02-02"></a>

#### [영상 90.3.2.35.a101.1.2.2. `도구 옵션` → `경로` → `편집 모드` → `편집`: 선분에 제어점 추가하기 (Windows) (우리말)](./90-03-02-35-paths.md#90-03-02-35-a101-01-02-02)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/9b387ff3-6d2d-4895-99bc-7f4e040df439"></video>

<a id="14-05-02-03-02-s5"></a>

## 5. 노드 지우기(Remove a node)
Shift(⇧) + Ctrl(⌘) 키를 누른채로, [노드](./19-glossaryx-path_node.md) 위로 마우스 포인터를 옮기면, 마우스 포인터가 `-` 기호로 바뀝니다. 

<a id="90-03-02-35-a201-05"></a>

#### [그림 90.3.2.35.a201.5. 마우스 포인터 아이콘(`-`)](./90-03-02-35-paths.md#90-03-02-35-a201-05)
![90-03-02-35-a201-05](https://github.com/wonder13662/gimp/assets/15767104/49bb494e-5981-4423-9922-82ff580966ff)

이 상태에서 [선분](./19-glossaryx-path_segment.md)을 클릭하면 선분이 지워집니다.

<a id="90-10-35-02-03-a1"></a>

#### [영상 90.10.35.2.3.a1. Ctrl(⌘) + Shift(⇧): 선분 지우기](./90-10-35-02-03-remove_segment.md#90-10-35-02-03-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/ce1ca37e-0556-45c2-bea2-a4f7cbb6f4e5"></video>

이 상태에서 [제어점](./19-glossaryx-path_control_point.md)을 클릭하면 제어점이 지워집니다.

<a id="90-10-35-02-04-a1"></a>

#### [영상 90.10.35.2.4.a2. Ctrl(⌘) + Shift(⇧): 제어점 지우기](./90-10-35-02-04-remove_control_point.md#90-10-35-02-04-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/e648c313-194c-4fc5-bd42-de7d7a29c556"></video>

<a id="14-05-02-03-02-s6"></a>

## 6. 노드에 핸들 추가하기(Add a handle to a node)
[제어점](./19-glossaryx-path_control_point.md)를 클릭합니다. 제어점을 드래그합니다. [핸들](./19-glossaryx-path_handle.md)이 나타납니다. 

<a id="90-03-02-35-a101-01-02-03"></a>

#### [영상 90.3.2.35.a101.1.2.3. `도구 옵션` → `경로` → `편집 모드` → `편집`: 제어점에 핸들 추가하기 (Windows) (우리말)](./90-03-02-35-paths.md#90-03-02-35-a101-01-02-03)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/d05f3749-161c-4802-a7ff-607c2ce05e8d"></video>

Shift(⇧)를 누르면 핸들을 대칭으로 움직일 수 있습니다.

<a id="90-10-35-04-01-a1"></a>

#### [영상 90.10.35.4.1.a1. Shift(⇧): 핸들을 대칭으로 움직이기 (Windows) (우리말)](./90-10-35-04-01-move_handle_symmetrically.md#90-10-35-04-01-a1)

## 7. 노드의 핸들 지우기(Remove a handle from a node)
Ctrl(⌘) + Shift(⇧) 키를 누른채로, [핸들](./19-glossaryx-path_handle.md) 위로 마우스 포인터를 옮기면, 마우스 포인터가 `-` 기호로 바뀝니다. 

<a id="90-03-02-35-a201-05"></a>

#### [그림 90.3.2.35.a201.5. 마우스 포인터 아이콘(`-`)](./90-03-02-35-paths.md#90-03-02-35-a201-05)
