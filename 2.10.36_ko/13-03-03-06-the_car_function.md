# 13.3.3.6. `car` 함수
`car` 함수는 리스트의 첫번째 요소를 반환합니다. 

`car` 함수의 인자로 받는 리스트는 비어있지 않아야 합니다. 

그러므로, 아래 예시에서는 리스트의 첫번째 요소를 반환합니다. 

```scheme
(car '("first" 2 "third"))
```

`car` 함수의 결과는 아래와 같습니다.

```scheme
"first"
```
