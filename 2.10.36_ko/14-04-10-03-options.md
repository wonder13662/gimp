# 14.4.10.3. 옵션(Options)

<a id="90-03-02-11-a1"></a>

#### [그림 90.3.2.11.a1. `도구상자` → `통합 변형`의 도구옵션](./90-03-02-11-unified_transform.md#90-03-02-11-a1)
![90-03-02-11-a1](https://github.com/wonder13662/gimp/assets/15767104/d6eebbf0-c197-4c44-8f5a-5af5105bee1b)

사용자가 도구를 선택해서 활성화시키면 도구상자의 아래쪽에 도구옵션 대화상자가 나타납니다. 그렇지 않다면, 도구 옵션 대화상자가 사라진 것일 수 있습니다. 이런 경우에는 [4.3.2. 도구 옵션 대화상자가 사라졌습니다](./04-03-02-tool-options-dialog-is-missing.md)를 참고해주세요. 더 자세한 내용은 [3.2.3. 대화상자(Dialog)와 도킹(docking)](./03-02-03-00-dialogs-and-docking.md)을 확인해주세요.

변형 도구 옵션의 공통적인 기능을 확인하고 싶다면, [14.4.1.1. 도구 옵션(Tool Options)](./14-04-01-01-00-tool_options.md)을 참고해주세요.

## 8. 제한하기(Constrain)

<a id="14-04-10-03-s8-01"></a>

### 8-1. 이동(Move)
`이동(Move)` 옵션을 켜면, 대상을 이동시키는 방향이 45도 단위로 제한됩니다. `이동(Move)` 옵션을 끄면 각도의 제한없이 자유롭게 이동시킬 수 있습니다.

<a id="90-03-02-11-a10-01"></a>

#### [영상 90.3.2.11.a10.1. `도구 옵션` → `통합 변형` → `제한하기` → `이동`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-01)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/20ccf395-e957-4d25-a0ca-9d64bb48d245"></video>

<a id="14-04-10-03-s8-02"></a>

### 8-2. 크기 조정(Scale)
`크기 조정(Scale)` 옵션을 켜면, 대상의 현재의 종횡비 비율(Aspect ratio)이 고정됩니다.

<a id="90-03-02-11-a10-02"></a>

#### [영상 90.3.2.11.a10.2. `도구 옵션` → `통합 변형` → `제한하기` → `크기 조정`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-02)

<a id="14-04-10-03-s8-03"></a>

### 8-3. 회전(Rotate)
`회전(Rotate)` 옵션을 켜면, 대상을 회전시킬 때 15도씩 회전합니다.

<a id="90-03-02-11-a10-03"></a>

#### [영상 90.3.2.11.a10.3. `도구 옵션` → `통합 변형` → `제한하기` → `회전`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-03)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/1e99e3f2-e62f-4a18-ba55-b6c10550dcb8"></video>

<a id="14-04-10-03-s8-04"></a>

### 8-4. 기울이기(Shear)
`기울이기(Shear)` 옵션을 켜면, 대상을 기울일 때 선택한 핸들이 있는 변의 방향으로만 움직입니다.

<a id="90-03-02-11-a10-04-01"></a>

#### [그림 90.3.2.11.a10.4.1. `도구 옵션` → `통합 변형` → `제한하기` → `기울이기`: 옵션 켠 뒤의 이동방향 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-04-01)

<a id="90-03-02-11-a10-04"></a>

#### [영상 90.3.2.11.a10.4. `도구 옵션` → `통합 변형` → `제한하기` → `기울이기`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-04)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/5134d43b-99ac-405f-b233-0cb4db940fd1"></video>

### 8-5. 원근법(Perspective)
`원근법(Perspective)` 옵션을 켜면, 대상을 왜곡할 때 선택한 핸들이 걸쳐있는 두개의 변의 방향 또는 중심점과 클릭한 지점의 대각선의 방향으로만 움직입니다.

<a id="90-03-02-11-a10-05"></a>

#### [영상 90.3.2.11.a10.5. `도구 옵션` → `통합 변형` → `제한하기` → `원근법`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-05)

## 9. 회전축 기준(From pivot)

<a id="14-04-10-03-s9-01"></a>

### 9-1. 크기 조정(Scale)
`도구 옵션` → `통합 변형` → `회전축 기준(From pivot)` → `크기 조정` 옵션을 켜면, [피벗(회전축)](./14-04-10-00-unified-transform.md#14-04-10-s2)을 중심으로 대상의 크기를 조정합니다. 옵션을 끄면, 클릭, 드래그하는 핸들의 위치에 따라 대상의 크기를 조정합니다.

<a id="90-03-02-11-a10-06"></a>

#### [그림 90.3.2.11.a10.6. `도구 옵션` → `통합 변형` → `회전축 기준(From pivot)` → `크기 조정`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-06)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/0ff18d71-a1ab-4df8-9e56-9cd41f8d4eae"></video>

<a id="14-04-10-03-s9-02"></a>

### 9-2. 기울이기(Shear)
`도구 옵션` → `통합 변형` → `회전축 기준(From pivot)` → `기울이기` 옵션을 켜면, [피벗(회전축)](./14-04-10-00-unified-transform.md#14-04-10-s2)을 중심으로 대상을 기울입니다. 옵션을 끄면, 반대편의 변들은 고정되며, [피벗(회전축)](./14-04-10-00-unified-transform.md#14-04-10-s2)도 이에 따라 움직입니다.

<a id="90-03-02-11-a10-07"></a>

#### [그림 90.3.2.11.a10.7. `도구 옵션` → `통합 변형` → `회전축 기준(From pivot)` → `기울이기`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-07)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/4b796277-7a34-47f2-a26d-f76685253045"></video>

<a id="14-04-10-03-s9-03"></a>

### 9-3. 원근법(Perspective)
`도구 옵션` → `통합 변형` → `회전축 기준(From pivot)` → `원근법` 옵션을 켜면, [피벗(회전축)](./14-04-10-00-unified-transform.md#14-04-10-s2)의 위치가 유지됩니다.

<a id="90-03-02-11-a10-08"></a>

#### [그림 90.3.2.11.a10.8. `도구 옵션` → `통합 변형` → `회전축 기준(From pivot)` → `원근법`: 옵션 켜기/끄기 (Windows) (우리말)](./90-03-02-11-unified_transform.md#90-03-02-11-a10-08)

## 10. 회전축(Pivot)

<a id="14-04-10-03-s10-01"></a>

### 10-1. 달라붙기(Snap)
