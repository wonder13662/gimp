# 14.4.3.2. 동작 변경키(Key modifiers)

<a id="14-04-03-02-s1"></a>

## 1. Ctrl(⌘)
`도구 옵션` → `이동` → `이동` → `레이어`일 경우, Ctrl(⌘) 을 누른채로 경로를 옮길 수 있습니다.

#### [영상 90.10.23.2.1.a1. 변형 도구 `이동`: Ctrl(⌘) 키 - `도구 옵션` → `이동` → `이동` → `경로`로 변경](./90-10-23-02-01-switch_to_move_path.md#90-10-23-02-01-a1)

## 2. Alt(⌥)
`도구 옵션` → `이동` → `이동` → `레이어`일 경우, Alt(⌥) 를 누른채로 선택을 옮길 수 있습니다.

<a id="90-10-23-03-01-a1"></a>

#### [영상 90.10.23.3.1.a1. 변형 도구 `이동`: Alt(⌥) 키 - `도구 옵션` → `이동` → `이동` → `선택`으로 변경](./90-10-23-03-01-switch_to_move_selection.md#90-10-23-03-01-a1)

<a id="14-04-03-02-s3"></a>

## 3. Shift(⇧)

### 3-1. `도구 옵션` → `이동` → `이동` → `레이어`

|선택된 값|Shift(⇧)키를 누름|Shift(⇧)키를 놓음|
|---|---|---|
|`현재 레이어 이동`|`이동할 레이어나 안내선 선택`|`현재 레이어 이동`|
|`이동할 레이어나 안내선 선택`|`현재 레이어 이동`|`이동할 레이어나 안내선 선택`|

<a id="90-10-23-03-01-a1"></a>

#### [영상 90.10.23.4.1.a1. 변형 도구 `이동`: Shift(⇧) 키 - `도구 옵션` → `이동` → `이동` → `레이어`의 `이동할 레이어나 안내선 선택`과 `현재 레이어 이동`을 바꾸기](./90-10-23-04-01-toggle_tool.md#90-10-23-03-01-a1)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/5ea34602-a4da-407a-a720-7f6cb8ce3e75"></video>

### 3-2. `도구 옵션` → `이동` → `이동` → `경로`

|선택된 값|Shift(⇧)키를 누름|Shift(⇧)키를 놓음|
|---|---|---|
|`이동할 경로 고르기`|`현재 경로 이동`|`이동할 경로 고르기`|
|`현재 경로 이동`|`이동할 경로 고르기`|`현재 경로 이동`|

<a id="90-10-23-03-01-a2"></a>

#### [영상 90.10.23.4.1.a2. 변형 도구 `이동`: Shift(⇧) 키 - `도구 옵션` → `이동` → `이동` → `경로`의 `이동할 경로 고르기`와 `현재 경로 이동`을 바꾸기](./90-10-23-04-01-toggle_tool.md#90-10-23-03-01-a2)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/c38099fb-2059-41c2-8cfe-6ee939024b76"></video>

### 3-3. 방향키 + Shift(⇧)
`레이어`, `선택`, `경로`를 방향키로만 움직이면 한번에 1픽셀씩 움직입니다. 하지만 방향키 + Shift(⇧)키로 움직이면 한번에 25픽셀씩 움직입니다.
