## 2. 회전 중심점(Center X/Y)
`회전 중심점(Center X/Y)`은 회전 중심점의 X, Y 좌표를 설정할 수 있습니다. 회전 중심점은 십자선이 있는 원으로 표시됩니다.

<a id="14-04-05-04-s2-01"></a>

### 2-1. 좌표 입력하기
회전 중심점의 X, Y 좌표를 대화상자의 입력창에 직접 입력할 수 있습니다.

<a id="90-04-111-a112-01"></a>

#### [영상 90.4.111.a112.1. `회전 조정` 대화상자 → `회전 중심점`: 좌표 입력하기 (Windows) (우리말)](./90-04-0111-rotation_adjustment.md#90-04-111-a112-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/74cee55f-96c6-4eee-b259-e3efdc551c2d"></video>

<a id="14-04-05-04-s2-02"></a>

### 2-2. 중심점 선택하기
대화상자의 회전 중심점의 아이콘 중에 원하는 중심점 위치를 클릭해서 옮길 수 있습니다.

<a id="90-04-111-a112-02"></a>

#### [영상 90.4.111.a112.2. `회전 조정` 대화상자 → `회전 중심점`: 중심점 선택하기 (Windows) (우리말)](./90-04-0111-rotation_adjustment.md#90-04-111-a112-02)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/26b6befe-c30e-4cb1-a2b3-511712fbcf67"></video>

<a id="14-04-05-04-s2-03"></a>

### 2-3. 중심점 클릭, 드래그하기
캔버스에 표시된 변형 핸들의 중심점을 클릭, 드래그하여 회전 중심점을 옮길 수 있습니다.

<a id="90-04-111-a112-03"></a>

#### [영상 90.4.111.a112.3. `회전 조정` 대화상자 → `회전 중심점`: 중심점 클릭, 드래그하기 (Windows) (우리말)](./90-04-0111-rotation_adjustment.md#90-04-111-a112-03)

## 3. 초기화 버튼(Reset button)
버튼을 클릭하면, 회전 작업의 초기값으로 되돌려줍니다.

<a id="90-04-111-a113-01"></a>

#### [영상 90.4.111.a113.1. `회전 조정` 대화상자 → `초기화` 버튼 (Windows) (우리말)](./90-04-0111-rotation_adjustment.md#90-04-111-a113-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/9ce85771-e712-4bcd-bdb4-576d76966913"></video>

<a id="14-04-05-04-s4"></a>

## 4. 재조정 버튼(Readjust button)
이 버튼을 클릭하면, 회전된 이미지는 그대로 두고, 변형 핸들을 원래 위치와 각도로 되돌립니다. 여러 단계로 복잡한 변형을 할 때 유용합니다.

<a id="90-04-111-a114-01"></a>

#### [영상 90.4.111.a114.1. `회전 조정` 대화상자 → `재조정` 버튼 (Windows) (우리말)](./90-04-0111-rotation_adjustment.md#90-04-111-a114-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/0ec044ab-2488-40ac-a010-efb8781aef93"></video>

<a id="14-04-05-04-s5"></a>

## 5. 다른 방법으로 레이어 회전시키기
[`레이어` → `변형` → `임의 각도 회전`](./16-07-49-arbitrary-rotation.md)으로 레이어를 회전시킬 수 있습니다.
