# 7.9.1. 붓 크기를 바꾸는 방법
붓 크기를 여러가지 방법으로 바꿀 수 있습니다.

<a id="07-09-01-s1"></a>

## 1. 슬라이더로 붓 크기 바꾸기
도구 옵션의 `크기` 슬라이더를 사용해서 붓 크기를 바꿀 수 있습니다.

<a id="90-03-02-23-a114"></a>

#### [그림 90.3.2.23.a114. `도구 옵션` → `그리기 붓` → `크기`](./90-03-02-23-paintbrush.md#90-03-02-23-a114)
![90-03-02-23-a114](https://github.com/wonder13662/gimp/assets/15767104/184fcad8-9cac-4838-aefd-c76e731b0be8)

칠하기 도구 중에 그리기 붓, 연필, 에어브러시, 지우개, 복제(Clone), 원근 복제(Perspective Clone), 복원(Healing), 문지르기(Smudge), 흐릿하게/선명하게, 닷지/번은 도구 옵션의 `크기` 슬라이더를 쓸 수 있습니다.

<a id="90-03-a121"></a>

#### [그림 90.3.a121. 붓 대화상자를 사용하는 도구상자 아이콘](./90-03-00-toolbox.md#90-03-a121)
![90-03-a121](https://github.com/wonder13662/gimp/assets/15767104/3d227356-1a44-4a9b-95be-7912d864e348)

<a id="07-09-01-s2"></a>

## 2. 화살표 키로 붓 크기 바꾸기
`크기` 슬라이더를 클릭하여 포커스가 `크기` 슬라이더로 이동한 뒤에, 화살표 키로 붓 크기를 바꿀 수 있습니다. 한번 누를 때마다 1씩 증가/감소합니다. 계속 누르고 있으면 1씩 연속으로 빠르게 바뀝니다.

<a id="90-10-07-08-01-a1"></a>

#### [영상 90.10.7.8.1.a1. `도구 옵션` → `그리기 붓` → `크기`: 화살표 키로 붓 크기 바꾸기](./90-10-07-08-01-adjust_brush_size-arrow_key.md#90-10-07-08-01-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/84e23845-cf13-447f-9207-e7ebaaf5d2e1"></video>

화살표 키가 작동하지 않을 경우에 직접 설정하기 위해서는 [7.9.1. 붓 크기를 바꾸는 방법 - 7. 키보드 위, 아래 화살표키에 붓 크기 바꾸기 설정하기](./07-09-01-how-to-change-the-size-of-a-brush.md#07-09-01-s5)을 참고해주세요.

<a id="07-09-01-s3"></a>

## 3. Page Down, Page Up 키로 붓 크기 바꾸기
`크기` 슬라이더를 클릭하여 포커스가 `크기` 슬라이더로 이동한 뒤에, Page Down, Page Up 키로 붓 크기를 바꿀 수 있습니다. 한번 누를 때마다 10씩 증가/감소합니다. 계속 누르고 있으면 10씩 연속으로 빠르게 바뀝니다.

<a id="90-10-07-08-02-a1"></a>

#### [영상 90.10.7.8.2.a1. `도구 옵션` → `그리기 붓` → `크기`: Page Down, Page Up 키로 붓 크기 바꾸기](./90-10-07-08-02-adjust_brush_size-page_up_n_down.md#90-10-07-08-02-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/38119326-09e4-4b1a-b2e3-f85871e6d9f3"></video>

<a id="07-09-01-s4"></a>

## 4. 단축키로 붓 크기 바꾸기
붓 크기를 바꾸는 기본 단축키를 사용할 수 있습니다.

- 붓 크기를 1씩 줄이기: `[`
- 붓 크기를 1씩 키우기: `]`

<a id="90-10-07-08-03-a1"></a>

#### [영상 90.10.7.8.3.a1. `도구 옵션` → `그리기 붓` → `크기`: `[`, `]` 키로 붓 크기 바꾸기](./90-10-07-08-03-adjust_brush_size-bracket_key.md#90-10-07-08-03-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/50debe52-352d-414c-b81e-4a081319158c"></video>

- 붓 크기를 10씩 줄이기: `{`
- 붓 크기를 10씩 키우기: `}`

<a id="90-10-07-08-04-a1"></a>

#### [영상 90.10.7.8.4.a1. `도구 옵션` → `그리기 붓` → `크기`: `{`, `}` 키로 붓 크기 바꾸기](./90-10-07-08-04-adjust_brush_size-curly_bracket_key.md#90-10-07-08-04-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/93684f62-dd94-46dc-ad4a-6e0d3089b3d7"></video>

<a id="07-09-01-s5"></a>

## 5. 마우스 스크롤휠로 붓 크기 바꾸기
마우스 스크롤휠로 붓의 크기를 바꿀 수 있습니다.

- "Ctrl + Alt + 스크롤휠 내리기"로  1씩 붓의 크기를 줄입니다.
- "Ctrl + Alt + 스크롤휠 올리기"로  1씩 붓의 크기를 키웁니다.

<a id="90-10-07-06-01-a2"></a>

#### [영상 90.10.7.6.1.a2. Ctrl(⌘) + Alt(⌥) + 마우스 휠: 붓 크기 바꾸기](./90-10-07-06-01-adjust_tool_size.md#90-10-07-06-01-a2)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/713a231e-16c4-4b8a-ae6c-859ab3b016f6"></video>

⚠️ 주의: Mac에서는 마우스 스크롤휠로 붓 크기 바꾸기가 작동하지 않습니다.

<a id="07-09-01-s6"></a>

## 6. 마우스 휠에 붓 크기 바꾸기 설정하기

### 6-1. `기본 설정` → `입력 장치` → `입력 제어기`
`기본 설정` → `입력 장치` → `입력 제어기`를 선택합니다. **현재 사용중인 제어기** 목록에서 "Main Mouse Wheel"을 더블 클릭합니다.

<a id="90-02-08-01-a11"></a>

#### [90.2.8.1.a11. `기본설정` → `입력 장치` → `입력 제어기` (windows) (우리말) - Main Mouse Wheel](./90-02-08-01-input-controllers.md#90-02-08-01-a11)
![90-02-08-01-a11](https://github.com/wonder13662/gimp/assets/15767104/323465c7-7f4e-42f8-bf4f-e79fe10e5d6e)

### 6-2. 이벤트 편집 모드로 들어가기

입력 제어기 설정 창이 나옵니다. 왼쪽 컬럼 "이벤트(Event)"에서 "위로 스크롤(Scroll Up)"을 클릭해서 하이라이트 상태로 만듭니다. 창 아래쪽의 "이벤트 편집(Edit Event)" 버튼을 누릅니다.

<a id="90-04-57-a111"></a>

#### [그림 90.4.58.a111. `제어기의 이벤트 동작 선택` 대화상자 (Windows) (우리말) - 위로 스크롤](./90-04-0057-configure_input_controller.md#90-04-57-a111)
![90-04-57-a111](https://github.com/wonder13662/gimp/assets/15767104/d28846a0-8ded-4e2b-9f45-c572d8398035)

### 6-3. 제어기의 이벤트 동작 선택 대화상자에서 "도구 크기: 1증가" 설정하기

"제어기의 이벤트 동작 선텍" 창이 뜹니다. 왼쪽 "+" 버튼을 클릭해서 "도구" 항목을 펼칩니다.

#### 그림 7.9.1.a103. "도구" 항목 펼치기
![07-09-01-toggle-tools](https://github.com/wonder13662/gimp/assets/15767104/9ae77b6e-42fe-4e83-ad1d-a9d11c949a1a)

왼쪽 컬럼 "동작"에서 "도구 크기: 1증가"을 클릭해서 하이라이트 상태로 만듭니다. 그리고 "OK" 버튼을 클릭합니다.

<a id="90-04-58-a111"></a>

#### [그림 90.4.58.a111. `제어기의 이벤트 동작 선택` 대화상자 (Windows) (우리말) - 도구 크기: 1 증가](./90-04-0058-select_controller_event_action.md#90-04-58-a111)

![07-09-01-scroll_up-tools-size-increase(windows)(ko)-focus](https://github.com/wonder13662/gimp/assets/15767104/a394835d-aa22-474b-a771-d8da4e1e3fc1)

창을 닫습니다.

### 6-4. 제어기의 이벤트 동작 선택 대화상자에서 "도구 크기: 1감소" 설정하기
같은 방식으로 "아래로 스크롤(Scroll Down)"에 "도구 크기: 1감소"을 설정해주세요. 완료 후, `기본 설정` 창의 "확인" 버튼을 클릭해주세요
