# 7.2.2. 선택 더하기/빼기
도구에는 직접 설정할 수 있는 옵션이 있습니다. 각 선택 도구는 옵션 안에서 선택 모드를 정할 수 있습니다. 아래에 나온 선택 모드를 사용할 수 있습니다.

<a id="07-02-02-s1"></a>

## 1. 선택 모드:현재 선택 바꾸기(Replace)

<a id="90-04-01-03-a111"></a>

#### [그림 90.4.1.3.a111. 도구상자의 사각 선택의 도구옵션 - 선택 모드:현재 선택 바꾸기(Replace)](./90-04-01-03-rectangle_select.md#90-04-01-03-a111)
![90-04-01-03-mode-replace(windows)(ko)](https://github.com/wonder13662/gimp/assets/15767104/522ef08a-7878-4a19-a5e4-26e8f3572831)

`현재 선택 바꾸기(Replace)` 모드는 가장 많이 사용됩니다. 이 모드에서는 새로 만든 선택이 이전의 선택을 없애고 대신합니다.

<a id="90-04-01-03-a121"></a>

#### [영상 90.4.1.3.a121. `도구 옵션` → `사각 선택` → `모드`:현재 선택 바꾸기(Replace)](./90-04-01-03-rectangle_select.md#90-04-01-03-a121)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/3c61de0e-8173-4e64-a531-4f9c8fa3080f"></video>

<a id="07-02-02-s2"></a>

## 2. 선택 모드:현재 선택에 더하기(Add)

<a id="90-04-01-03-a112"></a>

#### [그림 90.4.1.3.a112. 도구상자의 사각 선택의 도구옵션 - 선택 모드:현재 선택에 더하기(Add)](./90-04-01-03-rectangle_select.md#90-04-01-03-a112)
![90-04-01-03-a112](https://github.com/wonder13662/gimp/assets/15767104/54d29b7a-1ee2-45dd-806e-13a43d7a14e8)

`현재 선택에 더하기(Add)` 모드는 새로운 선택을 이미 있는 선택에 추가합니다.

<a id="90-04-01-03-a122"></a>

#### [영상 90.4.1.3.a122. `도구 옵션` → `사각 선택` → `모드`:현재 선택에 더하기(Add)](./90-04-01-03-rectangle_select.md#90-04-01-03-a122)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/31a3604e-c552-4fe4-805a-2c90e22c42f8
"></video>

또한 Shift(⇧) 키를 누른 채로 유지하면 일시적으로 `현재 선택에 더하기(Add)` 모드로 바뀝니다. Shift(⇧) 키를 누른 채로 클릭 + 드래그 한 뒤에 엔터키를 누르면, 기존 선택에 새로운 선택을 추가할 수 있습니다.

<a id="90-04-01-03-a125"></a>

#### [영상 90.4.1.3.a125. Shift(⇧) 눌러 선택 모드:현재 선택에 더하기(Add)로 전환](./90-04-01-03-rectangle_select.md#90-04-01-03-a125)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/e2b0823e-0ed6-4efb-b1e0-a07fd79e6c09"></video>

<a id="07-02-02-s3"></a>

## 3. 선택 모드:현재 선택에 빼기(Subtract)

<a id="90-04-01-03-a113"></a>

#### [그림 90.4.1.3.a113. 도구상자의 사각 선택의 도구옵션 - 선택 모드:현재 선택에 빼기(Subtract)](./90-04-01-03-rectangle_select.md#90-04-01-03-a113)
![90-04-01-03-mode-subtract(windows)(ko)](https://github.com/wonder13662/gimp/assets/15767104/e5f77996-607d-41c0-8c0e-ca66a0b106cf)

`현재 선택에 빼기(Subtract)` 모드는 새로운 선택을 이미 있는 선택에서 지웁니다.

<a id="90-04-01-03-a123"></a>

#### [영상 90.4.1.3.a123. `도구 옵션` → `사각 선택` → `모드`:현재 선택에 빼기(Subtract)](./90-04-01-03-rectangle_select.md#90-04-01-03-a123)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/a40217fa-c45e-4cfd-b54f-defc71a331db"></video>

또한 Ctrl(⌘) 키를 누른 채로 유지하면 일시적으로 `현재 선택에 빼기(Subtract)` 모드로 바뀝니다. Ctrl(⌘) 키를 누른 채로 클릭 + 드래그 한 뒤에 엔터키를 누르면, 기존 선택에서 새로운 선택을 지울 수 있습니다.

<a id="90-04-01-03-a126"></a>

#### [영상 90.4.1.3.a126. Ctrl(⌘) 눌러 선택 모드:현재 선택에 빼기(Subtract)로 전환](./90-04-01-03-rectangle_select.md#90-04-01-03-a126)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/33d4202e-d83f-4660-9b5f-a3e01585f98d"></video>

## 4. 선택 모드:현재 선택과 교차하기(Intersect)

<a id="90-04-01-03-a114"></a>

#### [그림 90.4.1.3.a114. `도구 옵션` → `사각 선택` → `모드`:현재 선택과 교차하기(Intersect)](./90-04-01-03-rectangle_select.md#90-04-01-03-a114)
![90-04-01-03-mode-intersect(windows)(ko)](https://github.com/wonder13662/gimp/assets/15767104/b56ef663-fc6a-4691-a04a-adb588d6038f)

`현재 선택과 교차하기(Intersect)` 모드는 새로운 선택과 이미 있는 선택의 교차한 부분을 새로운 선택으로 만듭니다.

<a id="90-04-01-03-a124"></a>

#### [영상 90.4.1.3.a124. `도구 옵션` → `사각 선택` → `모드`:현재 선택과 교차하기(Intersect)](./90-04-01-03-rectangle_select.md#90-04-01-03-a124)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/a0c68f09-01b8-4e59-8656-2f6797a5a7db"></video>

또한 Shift(⇧) + Ctrl(⌘) 키를 누른 채로 유지하면 일시적으로 `현재 선택과 교차하기(Intersect)` 모드로 바뀝니다. Shift(⇧) + Ctrl(⌘) 키를 누른 채로 클릭 + 드래그 한 뒤에 엔터키를 누르면, 교차된 새로운 선택을 만들 수 있습니다.

<a id="90-04-01-03-a127"></a>

#### [영상 90.4.1.3.a127. Shift(⇧) + Ctrl(⌘) 눌러 선택 모드:현재 선택과 교차하기(Intersect)로 전환](./90-04-01-03-rectangle_select.md#90-04-01-03-a127)
<video controls="controls" width="720" src="https://github.com/wonder13662/gimp/assets/15767104/e7aa6955-7b2b-4281-ba2e-3d0d702fa14d"></video>
