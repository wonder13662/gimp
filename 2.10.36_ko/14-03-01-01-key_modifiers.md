# 14.3.1.1. 동작 변경키(Key modifiers)

## 1. Ctrl(⌘)
Ctrl(⌘) 키를 누르는 것은 모든 칠하기 도구에 특별한 효과를 줍니다.

### 1-1. 색상 추출 도구(Color Picker)로 바꾸기
[연필](./14-03-06-pencil.md), [칠하기 붓](./14-03-07-paintbrush.md), [에어브러시](./14-03-10-airbrush.md), [잉크](./14-03-11-ink.md)에서 Ctrl(⌘) 키를 누르면, "색상 추출 도구(Color Picker)" 모드로 바뀝니다. 이 모드에서 이미지의 픽셀에 클릭하면, GIMP(김프)의 전경색이 활성화된 레이어의 클릭한 지점의 색상으로 설정됩니다.

`닷지/번` 도구에서 Ctrl(⌘) 키를 누르면 `번` 모드로 바뀝니다. Ctrl(⌘) 키를 떼면 다시 `닷지` 모드로 돌아갑니다.

## 2. Shift(⇧) - 직선 그리기
Shift(⇧) 키를 누르고 있으면, 대부분의 칠하기 도구에서 `직선 모드`로 바뀝니다.

칠하기 도구에서 직선을 그리려면, 시작점을 클릭합니다. 그리고 Shift(⇧) 키를 누릅니다. Shift(⇧) 키를 계속 누르고 있는 동안에는 시작점과 현재 마우스 포인터의 위치와 연결된 가는 선을 볼 수 있습니다. 

Shift(⇧) 키를 계속 누른 채로 다시 클릭하면, 직선이 그려지는 것을 확인할 수 있습니다. 이 과정을 반복하면 직선이 연속으로 이어집니다.

<a id="90-10-07-02-01-a1"></a>

#### [영상 90.10.7.2.1.a1. `도구상자` → `칠하기 붓` + Shift(⇧): 직선그리기](./90-10-07-02-01-draw_straight_line.md#90-10-07-02-01-a1)

## 3. Ctrl(⌘) + Shift(⇧)
Ctrl(⌘)키와 Shift(⇧) 키를 함께 누르고 있으면 칠하기 도구에서 `제한된 직선 모드`가 됩니다. `제한된 직선 모드`는 `직선 모드`와 비슷한 효과를 냅니다. 하지만 `제한된 직선 모드`는 직선의 방향이 15도씩만 회전할 수 있도록 강제된 것이 다릅니다 `제한된 직선 모드`로 완전한 수평, 수직, 대각선을 그릴 수 있습니다.

<a id="90-10-07-03-01-a1"></a>

#### [영상 90.10.7.3.1.a1. `도구상자` → `칠하기 붓` + Ctrl(⌘) + Shift(⇧): 직선그리기](./90-10-00-keyboard_shortcut.md#90-10-07-03-01-a1)
