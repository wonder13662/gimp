# 5.2. 새 파일 만들기
[이미지 메뉴 바]()의 `파일` → `새 이미지`를 선택해서 `새 이미지 만들기` 대화상자를 엽니다. 대화상자에서 너비와 높이를 조정하거나 초기값을 사용하여 새로운 이미지 파일을 만듭니다. `새 이미지 만들기` 대화상자에 대한 좀 더 많은 정보는 [16.2.2. 새 이미지](./16-02-02-new.md)를 확인해주세요.
