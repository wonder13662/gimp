# 5.2. 새 파일 만들기
[이미지 메뉴 바](./19-glossaryx-image_menu_bar.md)의 `파일` → `새 이미지`를 선택해서 `새 이미지 만들기` 대화상자를 엽니다. 

<a id="90-01-01-01-a101"></a>

#### [그림 90.1.1.1.a101. `이미지 메뉴 바` → `파일` → `새 이미지`](./90-01-01-01-new.md#90-01-01-01-a101)
