# 14.5.6. 텍스트 도구(Text)

<a id="90-03-02-36-a2"></a>

#### [그림 90.3.2.36.a2. `도구상자` → `텍스트`](./90-03-02-36-text.md#90-03-02-36-a2)
![90-03-02-36-a2](https://github.com/wonder13662/gimp/assets/15767104/6048b169-9886-400b-808c-5f46c40854ae)

텍스트 도구는 텍스트를 새로운 레이어에 추가합니다. 텍스트는 캔버스에서 직접 고칠 수 있습니다.

<a id="14-05-06-00-s1"></a>

## 1. 텍스트 도구 상자

<a id="90-04-67-a101"></a>

#### [그림 90.4.67.a101. `텍스트 도구상자` 대화상자](./90-04-67-text_toolbox.md#90-04-67-a101)
![90-04-67-a101](https://github.com/wonder13662/gimp/assets/15767104/103ac210-7099-4af7-85b3-e745b9fc7a42)

[`텍스트 도구 상자`](./09-01-03-00-text_toolbox.md)는 [텍스트 상자(사각형 프레임)](./14-05-06-00-text.md#14-05-06-00-s5) 위에 캔버스 위쪽으로 나타납니다.

[관련 정보: 9.1.3. 텍스트 도구상자(Text Toolbox)](./09-01-03-00-text_toolbox.md)

`텍스트 도구 상자`는 선택한 텍스트나 새로 입력하는 텍스트에 대해 텍스트 속성을 바로 바꿀수 있도록 해줍니다.

<a id="90-04-67-a161"></a>

#### [영상 90.4.67.a161. `텍스트 도구상자` 대화상자 - 선택한 텍스트의 색 바꾸기](./90-04-67-text_toolbox.md#90-04-67-a161)

## 2. GIMP 텍스트 편집기(GIMP Text Editor)
`GIMP 텍스트 편집기(GIMP Text Editor)`를 사용하지 않고도 텍스트를 수정할 수 있지만, `도구 옵션` → `텍스트` → `편집기 사용` 옵션을 켜면 `GIMP 텍스트 편집기(GIMP Text Editor)`를 사용할 수 있습니다.

[관련 정보: 9.1.1. 텍스트 영역(Text Area) - 4. 김프 텍스트 편집기(GIMP Text Editor)로 텍스트 수정하기](./09-01-01-text_area.md#09-01-01-s4)

## 3. `도구 옵션` → `텍스트`

<a id="90-03-02-36-a1"></a>

#### [그림 90.3.2.36.a1. `도구상자` → `텍스트`의 도구옵션](./90-03-02-36-text.md#90-03-02-36-a1)
![90-03-02-36-a1](https://github.com/wonder13662/gimp/assets/15767104/bff908e8-6196-45cb-a10c-5afa768e8684)

`도구 옵션` → `텍스트`의 옵션들을 사용하여 텍스트의 글꼴, 색상, 크기, 정렬을 즉시 바꿀 수 있습니다.

<a id="14-05-06-00-s4"></a>

## 4. 텍스트 컨텍스트 메뉴
<a id="90-04-68-a101"></a>

#### [그림 90.4.68.a101. `텍스트 컨텍스트 메뉴` (Windows) (우리말)](./90-04-68-00-text_context_menu.md#90-04-68-a101)
![90-04-68-a101](https://github.com/wonder13662/gimp/assets/15767104/30c207a9-7939-4b4c-8635-e9760639d2ee)

텍스트 위에서 오른쪽 클릭을 하면 [텍스트 컨텍스트 메뉴](./09-01-04-00-text_context_menu.md)를 열 수 있습니다. 이 메뉴로 복사, 잘라내기, 붙여넣기, 텍스트 파일 열기 등의 작업을 할 수 있습니다.

[관련 정보: 9.1.4. 텍스트 컨텍스트 메뉴(Text Context Menu)](./09-01-04-00-text_context_menu.md)

<a id="14-05-06-00-s5"></a>

## 5. 텍스트 상자(사각형 프레임)
텍스트를 입력하자마자, 사각형 프레임이 캔버스에 나타납니다.

<a id="90-04-03-14-a61"></a>

#### [영상 90.4.3.14.a61. 텍스트 입력에 따라 텍스트 상자가 커집니다](./90-04-03-14-text_layer.md#90-04-03-14-a61)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/8c29a051-d425-4457-bb74-1fd570f57bee"></video>

사각형 프레임의 크기를 클릭, 드래그로 바꾸면, 텍스트가 사각형 프레임에 맞게 자동으로 수정됩니다. 사각형 프레임의 수정방법은 [7.2.1. 선택 옮기기와 크기 조정하기 - 1-1. 선택의 크기 바꾸기](./07-02-01-00-moving-or-resizing-a-selection.md#07-02-01-s1-01)와 같습니다.

<a id="90-04-03-14-a71"></a>

#### [영상 90.4.3.14.a71. 클릭, 드래그로 텍스트 상자 크기 바꾸기](./90-04-03-14-text_layer.md#90-04-03-14-a71)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/71a150b3-fdbf-4b6e-bc0d-deadb1777067"></video>
