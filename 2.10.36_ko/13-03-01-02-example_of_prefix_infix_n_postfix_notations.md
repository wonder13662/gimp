# 13.3.1.2. 접두사, 삽입사, 접미사 예제(Examples Of Prefix, Infix, And Postfix Notations)
다음과 같은 계산식을 접두사(Prefix), 삽입사(Infix), 접미사(Postfix) 방식으로 표현해 보겠습니다.
```
1 + 23 = 24
```

## 1. 접두사(Prefix)
`Scheme`에서 사용하는 방식입니다.

```
+ 1 23
```

## 2. 삽입사(Infix)
일상적으로 계산을 표현하는 방식입니다.

```
1 + 23
```

## 3. 접미사(Postfix)
HP 계산기에서 사용하는 방식입니다.

```
1 23 +
```
