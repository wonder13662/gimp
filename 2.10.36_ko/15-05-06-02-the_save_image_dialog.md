## 1. 이미지 파일 이름 설정하기

<a id="90-04-55-a131"></a>

#### [그림 90.4.55.a131. `이미지 저장` 대화상자 → `이름` 입력창 (Windows) (우리말)](./90-04-0055-save_image.md#90-04-55-a131)

## 2. 이미지 파일 저장 위치 설정하기

<a id="90-04-55-a141"></a>

#### [그림 90.4.55.a141. `이미지 저장` 대화상자 → `폴더에 저장` (Windows) (우리말)](./90-04-0055-save_image.md#90-04-55-a141)

#### [영상 90.4.55.a141.1. `이미지 저장` 대화상자 → `폴더에 저장`: 폴더 바꾸기 (Windows) (우리말)](./90-04-0055-save_image.md#90-04-55-a141-01)

## 3. 이미지를 저장할 새로운 폴더 추가하기

<a id="90-04-55-a151"></a>

#### [그림 90.4.55.a151. `이미지 저장` 대화상자 → `폴더 만들기` (Windows) (우리말)](./90-04-0055-save_image.md#90-04-55-a151)

#### [영상 90.4.55.a151.1. `이미지 저장` 대화상자 → `폴더 만들기` (Windows) (우리말)](./90-04-0055-save_image.md#90-04-55-a151-01)

## 4. 파일 유형 선택(Select File Type)
대화상자의 아래쪽의 `파일 유형 선택(Select File Type)` 옵션을 클릭해서 펼치면, [`XCF`](./19-glossaryx-xcf.md) 포맷과 [`XCF`](./19-glossaryx-xcf.md)의 압축파일 포맷을 선택할 수 있습니다.
