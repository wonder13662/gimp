# 15.4.4.2.4.2.5. 이 이미지의 색상 관리하기(Color manage this image)

<a id="90-04-149-a145"></a>

#### [그림 90.4.149.a145. `양식 편집` 대화상자 → `고급 옵션` → `이 이미지의 색상 관리하기` (Windows) (우리말)](./90-04-0149-edit_template.md#90-04-149-a145)
<img width="479" height="684" alt="90-04-149-a145" src="https://github.com/user-attachments/assets/311bb141-20a1-40e9-893e-da790a935848" />

`이 이미지의 색상 관리하기(Color manage this image)` 옵션이 켜지면, 이 이미지는 색상 관리를 하게 됩니다.

<a comment="'색상관리'의 정확한 역할을 파악하여 이해하기 쉽도록 바꿔야 한다."></a>

이 옵션이 꺼지면, [15.4.4.2.4.2.6. 색상 프로파일(Color Profile)](./15-04-04-02-04-02-06-color_profile.md)에서 기본값인 `Built-in RGB(GIMP built-in sRGB)`를 선택한 것과 같습니다.

<a comment="Gitlab의 이 옵션의 역할에 대해서 정확한 설명을 요청해야 한다."></a>
