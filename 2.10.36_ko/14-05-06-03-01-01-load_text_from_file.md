# 14.5.6.3.1.1. 파일에서 텍스트 불러오기 (Load text from file)

<a id="90-04-65-a203-01"></a>

#### [그림 90.4.65.a203.1. 김프 텍스트 편집기 대화상자: 파일에서 텍스트 불러오기 (Windows) (우리말)](./90-04-65-gimp_text_editor.md#90-04-65-a203-01)
<img width="306" height="260" alt="90-04-65-a203-01" src="https://github.com/wonder13662/gimp/assets/15767104/26441e16-1f9f-4679-858c-e1e5454f5681" />

텍스트를 파일로부터 불러올 수 있습니다. 

`김프 텍스트 편집기 대화상자`의 위쪽의 폴더 아이콘을 클릭해서 파일을 선택하여 텍스트를 불러옵니다. 

파일의 모든 텍스트가 불러와집니다.
