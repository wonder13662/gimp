## 2. 마칠 때 도구 옵션 저장(Save tool options on exit)
도구 옵션의 변경사항을 GIMP(김프)를 종료할 때 저장합니다.

## 3. 도구 옵션 저장(Save Tool Options Now)
도구 옵션의 변경사항을 지금 바로 저장합니다.

## 4. 저장된 도구 옵션을 기본값으로 초기화(Reset Save Tool Options to Default Values)
도구 옵션의 모든 값을 초기화합니다.
