# 15.2.3.2.5. 미리보기 이미지(Preview Image)

<a id="90-04-05-a141-05"></a>

#### [그림 90.4.5.a141.5. `경로 대화상자` → `미리보기 이미지` (windows) (우리말)](./90-04-0005-paths.md#90-04-05-a141-05)
<img width="200" height="257" alt="90-04-05-a141-05" src="https://github.com/wonder13662/gimp/assets/15767104/9c4f1406-e2f4-4f95-9f0b-6be58e558221" />

`미리보기 이미지(Preview Image)`는 활성화된 경로에 대한 작은 미리보기 이미지를 보여줍니다.

`미리보기 이미지(Preview Image)`를 클릭, 드래그해서 이미지로 가져가면, 현재 이미지에 경로의 복사본을 만들어 줍니다.

<a id="90-04-05-a141-05-01"></a>

#### [영상 90.4.5.a141.5.1. `경로 대화상자` → `미리보기 이미지`: 미리보기를 클릭한 뒤, 이미지로 드래그하여 경로 복사하기 (windows) (우리말)](./90-04-0005-paths.md#90-04-05-a141-05-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/c81be7ec-2dc7-4b09-a173-17edd4858bdd"></video>

마우스 포인터를 미리보기 위에 올리고 오른쪽 클릭을 유지하고 있으면 미리보기가 확대되어 보입니다.

<a id="90-04-05-a141-05-02"></a>

#### [영상 90.4.5.a141.5.2. `경로 대화상자` → `미리보기 이미지`: 미리보기를 클릭 유지하여 미리보기 확대하기 (windows) (우리말)](./90-04-0005-paths.md#90-04-05-a141-05-02)
