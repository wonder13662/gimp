# 15.2.2.2.3.4. 채널 이름(Channel Name)

<a id="90-04-04-a11-04-04"></a>

#### [그림 90.4.4.a11.4.4. `채널 대화상자` → `채널 속성` → `채널 이름` (Windows) (우리말)](./90-04-04-channels.md#90-04-04-a11-04-04)
<img width="262" height="290" alt="90-04-04-a11-04-04" src="https://github.com/wonder13662/gimp/assets/15767104/ec055898-7206-4bb2-bc0e-dc55967fc65b" />

채널의 이름을 나타냅니다.

선택 마스크 채널의 이름을 더블 클릭하면, 이름을 직접 수정할 수 있습니다.

채널의 이름은 이미지 안에서 고유해야 합니다. 같은 이름을 강제로 쓸 경우, 접미사로 번호가 붙습니다.

<a id="90-04-04-a11-04-04-01"></a>

#### [영상 90.4.4.a11.4.4.1. `채널 대화상자` → `채널 속성` → `채널 이름`: 중복된 채널 이름에는 맨 뒤에 번호가 붙습니다 (Windows) (우리말)](./90-04-04-channels.md#90-04-04-a11-04-04-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/798680d8-48b6-423b-b168-05654e5483f1"></video>

원색 채널과 알파 채널의 이름은 수정할 수 없습니다.

미리보기를 더블 클릭해서 `채널 속성` 대화상자를 열어 채널의 속성인 이름을 수정할 수 있습니다.

아래 영상에서 `채널 미리보기`를 더블 클릭해서 `채널 속성` 대화상자를 열어 채널의 속성인 이름을 바꾸는 과정을 확인할 수 있습니다.

<a id="90-04-122-a101-01"></a>

#### [영상 90.4.122.a101.1. `채널 속성` 대화상자 → `채널 이름`: 이름 바꾸기 (Windows) (우리말)](./90-04-122-channel_attributes.md#90-04-122-a101-01)
