## 1. X와 Y
X와 Y의 값을 설정해서 [레이어](./19-glossaryx-layer.md)의 컨텐츠가 가로 방향(X)과 세로 방향(Y)으로 이동하는지 설정할 수 있습니다.

<a id="16-07-50-02-02-s1-01"></a>

### 1-1. 입력창에 값 입력하기
입력창과 슬라이더를 사용해서 X와 Y값을 설정할 수 있습니다.

X의 양의 값은 레이어를 오른쪽으로 움직입니다.

Y의 양의 값은 레이어를 아래쪽으로 움직입니다.

<a id="90-04-195-a122"></a>

#### [영상 90.4.195.a122. `오프셋` 대화상자 → `오프셋`: 입력창에 값 입력하기 (Windows) (우리말)](./90-04-0195-offset.md#90-04-195-a122)

<a id="16-07-50-02-02-s1-02"></a>

### 1-2. 단위 바꾸기
기본 단위값은 픽셀입니다.

그리고 드롭다운 목록에서 다른 단위를 선택할 수 있습니다.

<a id="90-04-195-a123"></a>

#### [그림 90.4.195.a123. `오프셋` 대화상자 → `오프셋` → `단위`: 드롭다운 목록  (Windows) (우리말)](./90-04-0195-offset.md#90-04-195-a123)

### 1-3. 레이어를 클릭, 드래그해서 X, Y값 설정
레이어를 직접 클릭, 드래그해서 옮겨서 X와 Y값을 설정할 수 있습니다.

<a id="90-04-195-a124"></a>

## 2. `By width/2, height/2` 버튼 누르기
`By width/2, height/2` 버튼을 누르면, X의 오프셋이 이미지 너비의 절반, Y의 오프셋이 이미지 높이의 절반으로 설정됩니다.

<a id="90-04-195-a125"></a>

#### [영상 90.4.195.a125. `오프셋` 대화상자 → `오프셋`: `By width/2, height/2` 버튼 누르기 (Windows) (우리말)](./90-04-0195-offset.md#90-04-195-a125)
