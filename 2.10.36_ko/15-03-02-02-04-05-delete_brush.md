# 15.3.2.2.4.5. 붓 삭제(Delete Brush)

<a id="90-04-15-a101-01-05"></a>

#### [그림 90.4.15.a101.1.5. `붓` 대화상자 → 아래쪽 버튼들과 슬라이더 → `붓 삭제` (Windows) (우리말)](./90-04-0015-brushes.md#90-04-15-a101-01-05)
<img width="200" height="273" alt="90-04-15-a101-01-05" src="https://github.com/user-attachments/assets/263ba47f-e89f-4880-bbfa-d1e3af04e231" />

`붓` 대화상자 → 아래쪽 버튼들과 슬라이더 → `붓 삭제` 버튼은 [파라매트릭 붓](./07-06-05-paramatric_brush.md)에서만 사용할 수 있습니다.

`붓 삭제` 버튼은 `붓` 대화상자와 [사용자 붓 폴더](./12-01-25-data-folders.md)에서 붓을 삭제합니다.

`붓 삭제` 버튼을 누르면 정말 삭제할 것인지 확인하는 `객체 삭제` 대화상자가 뜹니다.

<a id="90-04-133-a101"></a>

#### [그림 90.4.133.a101. `객체 삭제` 대화상자: 붓 (Windows) (우리말)](./90-04-0133-delete_object.md#90-04-133-a101)
<img width="394" height="174" alt="90-04-133-a101" src="https://github.com/user-attachments/assets/45f5fb86-418f-4ada-8e61-7130830e0595" />

[다른 운영체제와 언어의 `객체 삭제` 대화상자 확인하기](./90-04-0133-delete_object.md#90-04-133-a102)

이 대화상자에서 확인 버튼을 눌러야 실제로 붓이 삭제됩니다.
