# 15.5.1.3.5. 파일 관리자에서 도구 프리셋 파일 보기(Show in File Manager)
`파일 관리자에서 도구 프리셋 파일 보기(Show in File Manager)` 메뉴를 누르면, 선택한 도구 프리셋 파일을 시스템의 기본 파일 관리자를 열어 표시해줍니다.

<a id="15-05-01-03-05-s1"></a>

## 1. 도구 프리셋 대화상자 → `탭 메뉴` → `도구 프리셋 메뉴` → `파일 관리자에서 도구 프리셋 파일 보기`
