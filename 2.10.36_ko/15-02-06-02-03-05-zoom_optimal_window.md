# 15.2.6.2.3.5. 창 전체를 사용할 수 있도록 확대 비율을 조정(Adjust the zoom ratio so that the window is used optimally)

<a id="90-04-09-a101-02-03-05"></a>

#### [그림 90.4.9.a101.2.3.5. `내비게이션` 대화상자 → `창 전체를 사용할 수 있도록 확대 비율을 조정` (Windows) (우리말)](./90-04-0009-navigation.md#90-04-09-a101-02-03-05)

`내비게이션` 대화상자 → `창 전체를 사용할 수 있도록 확대 비율을 조정` 버튼을 누르면 이미지가 이미지 창에 남은 공간이 없이 표시됩니다.

이미지와 이미지 창의 종횡비 비율이 다르다면, 이미지의 일부가 이미지 창 바깥으로 나갈 수 있습니다.
