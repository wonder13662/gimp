# 16.6.23. 안내선으로 잘라내기(Slice Using Guides)

`안내선으로 잘라내기(Slice Using Guides)` 명령은 이미지를 안내선을 기준으로 자릅니다.

이 명령은 이미지를 각 안내선에 따라 자릅니다.

그리고 각 조각별로 새로운 이미지를 새로 만듭니다.
