# 14.3.14.3. 옵션(Options)

<a id="90-03-02-30-a1"></a>

#### [그림 90.3.2.30.a1. `도구상자` → `원근 복제`의 도구옵션](./90-03-02-30-perspective_clone.md#90-03-02-30-a1)
![90-03-02-30-a1](https://github.com/wonder13662/gimp/assets/15767104/8864cfcc-027c-486b-b7ce-3a607d20e545)

사용자가 도구를 선택해서 활성화시키면 도구상자의 아래쪽에 도구옵션 대화상자가 나타납니다. 그렇지 않다면, 도구 옵션 대화상자가 사라진 것일 수 있습니다. 이런 경우에는 [4.3.2. 도구 옵션 대화상자가 사라졌습니다](./04-03-02-tool-options-dialog-is-missing.md)를 참고해주세요. 더 자세한 내용은 [3.2.5. 대화상자(Dialog)와 도킹(docking)](./03-02-05-00-dialogs-and-docking.md)을 확인해주세요.

칠하기 도구 옵션의 공통적인 기능을 확인하고 싶다면, [14.3.1.2. 도구 옵션(Tool Options)](./14-03-01-02-00-tool_options.md)을 참고해주세요.

<a id="14-03-14-03-s1"></a>

## 1. 작동 모드(Operating mode)

<a id="14-03-14-03-s1-01"></a>

### 1-1. 원근 조정(Modify Perspective)
`도구 옵션` → `원근 복제` → `원근 조정` 옵션을 켜고 [원근법(Perspective)](./14-04-08-00-perspective.md) 도구와 같은 방법으로 투시를 위한 소멸선(vanishing line)을 설정합니다.

<a id="90-03-02-30-a111-01"></a>

#### [영상 90.3.2.30.a111.1. `도구 옵션` → `원근 복제` → `원근 조정` (Windows) (우리말)](./90-03-02-30-perspective_clone.md#90-03-02-30-a111-01)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/309a2bf4-5e6b-48e6-882b-36b748e9493f"></video>

<a id="14-03-14-03-s1-02"></a>

### 1-2. 원근 복제(Perspective Clone)
`도구 옵션` → `원근 복제` → `원근 복제` 옵션을 켜고 [도장(Clone)](./14-03-12-00-clone.md) 도구와 같은 방법으로 원본 영역을 복사합니다.

<a id="90-03-02-30-a111-02"></a>

#### [영상 90.3.2.30.a111.2. `도구 옵션` → `원근 복제` → `원근 복제` (Windows) (우리말)](./90-03-02-30-perspective_clone.md#90-03-02-30-a111-02)
