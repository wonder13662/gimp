# 15.3.4.2.3.3. 파일 관리자에 표시(Show in File Manager)

<a id="90-04-18-a113-03"></a>

#### [그림 90.4.18.a113.3. 무늬 대화상자 → `탭 메뉴` → `무늬 메뉴` → `파일 관리자에 표시` (Windows) (우리말)](./90-04-0018-patterns.md#90-04-18-a113-03)
