# 14.5.5.3. 동작 변경키(Key modifiers)

<a id="14-05-05-02-s1"></a>

## 1. Shift(⇧)
Shift(⇧) 키를 누른 채로, 끝점(Endpoint)로부터 클릭, 드래그를 하면 두번째 측정선을 그을 수 있고, 두 선분 간의 각도가 표시됩니다.

이를 이용해서 이미지의 어떤 곳이라도 각도를 잴 수 있습니다.

<a id="90-10-38-04-01-a1"></a>

#### [영상 90.10.38.4.1.a1. Shift(⇧): 두번째 측정선 그리기](./90-10-38-04-01-draw_second_measurement_line.md#90-10-38-04-01-a1)

<a id="14-05-05-02-s2"></a>

## 2. Ctrl(⌘)

<a id="14-05-05-02-s2-01"></a>

### 2-1. 측정선 회전 각도 15도로 제한
Ctrl(⌘) 키를 누르고 클릭, 드래그를 하면 측정선(Measurement line)을 15도씩 회전하도록 제한할 수 있습니다.

<a id="90-10-38-02-01-a1"></a>

#### [영상 90.10.38.2.1.a1. Ctrl(⌘): 측정선 회전 각도 15도로 제한 (Windows) (우리말)](./90-10-38-02-01-constrain_rotation_angle.md#90-10-38-02-01-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/5bdae857-5576-4891-9024-473b35951f48"></video>

<a id="14-05-05-02-s2-02"></a>

### 2-2. 수평 안내선 추가
Ctrl(⌘) 키를 누르고 끝점(Endpoint)를 클릭하면, 수평 방향의 안내선(Guide)가 추가됩니다.

<a id="90-10-38-02-02-a1"></a>

#### [영상 90.10.38.2.2.a1. Ctrl(⌘): 수평 안내선 추가 (Windows) (우리말)](./90-10-38-02-02-add_horizontal_guide.md#90-10-38-02-02-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/3e741673-82fd-4448-a171-2ff3d8aefafd"></video>

### 2-3. `수직`, `수평` 전환
`도구 옵션` → `방향` → `수직` 또는 `수평`을 선택한 경우, Ctrl(⌘) 키를 누르면 `수직`과 `수평`이 전환됩니다.

<a id="90-10-38-02-03-a1"></a>

#### [영상 90.10.38.2.3.a1. Ctrl(⌘): `수직`, `수평` 전환 (Windows) (우리말)](./90-10-38-02-03-toggle_horizontal_n_vertical.md#90-10-38-02-03-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/dad859c2-020c-415a-ad78-75ef38f46103"></video>

<a id="14-05-05-02-s3"></a>

## 3. Alt(⌥)
Alt(⌥)키를 누른 채로 끝점(Endpoint)를 클릭하면, 수직 안내선이 추가됩니다.

<a id="90-10-38-03-01-a1"></a>

#### [영상 90.10.38.3.1.a1. Alt(⌥): 수직 안내선 추가 (Windows) (우리말)](./90-10-38-03-01-add_vertical_guide.md#90-10-38-03-01-a1)

<a id="14-05-05-02-s4"></a>

## 4. Ctrl(⌘) + Alt(⌥)

<a id="14-05-05-02-s4-01"></a>

### 4-1. 측정 옮기기
Ctrl(⌘) + Alt(⌥)을 누르고 측정선을 클릭하면 측정을 옮길 수 있습니다.

<a id="90-10-38-03-02-a1"></a>

#### [영상 90.10.38.3.2.a1. Ctrl(⌘) + Alt(⌥): 측정 옮기기 (Windows) (우리말)](./90-10-38-03-02-move_measurement.md#90-10-38-03-02-a1)
<video controls="controls" width="640" height="360" src="https://github.com/wonder13662/gimp/assets/15767104/4e0b7437-e4b7-4f35-a0a0-e8806a3fcb37"></video>

<a id="14-05-05-02-s4-02"></a>

### 4-2. 수평, 수직 안내선 한번에 추가하기
Ctrl(⌘) + Alt(⌥)을 누르고 끝점을 클릭하면 수평, 수직 안내선을 모두 한번에 추가할 수 있습니다.

<a id="90-10-38-03-03-a1"></a>

#### [영상 90.10.38.3.3.a1. Ctrl(⌘) + Alt(⌥): 수평, 수직 안내선 한번에 추가하기 (Windows) (우리말)](./90-10-38-03-03-add_cross_guide.md#90-10-38-03-03-a1)
