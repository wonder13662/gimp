# 16.2.18. 파일 관리자에 표시(Show in File Manager)

`파일 관리자에 표시(Show in File Manager)` 메뉴를 누르거나 단축키 Ctrl(⌘) + Shift(⇧) + `F`을 누르면 운영체제의 파일 관리자가 열리며 작업중인 이미지 파일의 디렉토리를 보여줍니다.
