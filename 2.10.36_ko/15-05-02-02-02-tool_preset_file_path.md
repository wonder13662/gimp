# 15.5.2.2.2. 수정된 도구 프리셋이 저장되는 경로(Tool Preset File Path)
도구 프리셋을 수정하여 저장하면, `기본 설정` → `폴더` → `도구 프리셋`의 쓰기가능(Writable)한 첫번째 경로의 폴더에 저장됩니다.
