`페인트 동적 요소 편집기` 대화상자의 주요한 역할은 마우스 또는 태블릿 스타일러스(펜)에 영향을 미치는 붓 파라미터(매개변수)를 설정하는 것입니다. 원하는 만큼 파라미터들을 활성화시킬 수 있지만, 활성화시키는 파라미터가 적을 수록 더 좋습니다.

테이블의 각 행(컬럼)은 `무작위(Random)`, `점차 사라짐(Fade)`을 제외하고 마우스 또는 태블릿 스타일러스(펜)의 동작을 의미합니다. 

모든 파라미터(함수)는 그래픽 태블릿에서 동작합니다. 파라미터(함수)의 일부는 마우스로도 동작합니다. 
These functions are marked in the tables. The descriptions are using the default settings of all functions.

## 1. 열(Column)의 항목 설명 - 장치 동작

|항목|설명|
|---|---|
|압력(Pressure)|스타일러스가 태블릿에 주는 압력|
|속도(Veolcity)|(마우스 사용 가능)붓의 이동 속도|
|방향(Direction)|(마우스 사용 가능)붓의 이동 방향|
|기울기(Tilt)|스타일러스가 태블릿과 이루는 기울기|
|휠/회전(Wheel)|스타일러스의 회전값 또는 에어브러시 펜의 휠 설정|
|무작위(Random)|(마우스 사용 가능)선택된 행(row)의 옵션이 무작위 값으로 바뀝니다|
|점차 사라짐(Fade)|(마우스 사용 가능)선택된 행(row)의 옵션이 `도구 옵션` → `동적 요소 옵션` → `페이드 옵션`에 설정된 값에 따라 점차 사라짐이 적용됩니다|

## 2. 행(Row)의 항목 설명 - 붓 파라미터
각 행(Row)은 하나의 붓 파라미터와 장치 동작을 나타내는 7개의 체크박스로 구성되어 있습니다. 장치 동작의 체크박스를 클릭해서 파라미터와 장치 동작을 연결할 수 있습니다. 체크된 박스를 다시 클릭하면 체크가 해제됩니다.

### 2-1. 불투명도(Opacity)

|항목|설명|
|---|---|
|압력(Pressure)|압력을 더 세게 주면, 불투명도가 더 높게 그려집니다|
|속도(Veolcity)|(마우스 사용 가능)장치의 이동속도가 빨라지면, 불투명도가 감소합니다|
|방향(Direction)|(마우스 사용 가능)장치의 방향에 따라서 불투명도가 달라집니다. 방향에 따른 불투명도는 내장된 무작위 방식을 따릅니다.|
|기울기(Tilt)|불투명도가 스타일러스의 기울기에 따라 달라집니다|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|(마우스 사용 가능)불투명도가 `도구 옵션` → `불투명` 슬라이더의 값과 0 사이의 무작위 값으로 바뀝니다|
|점차 사라짐(Fade)|(마우스 사용 가능)완전 투명에서 시작해서 `도구 옵션` → `불투명` 슬라이더의 값으로 끝납니다|

### 2-3. 크기(Size)

|항목|설명|
|---|---|
|압력(Pressure)|압력을 더 세게 주면, 붓의 크기가 더 커집니다|
|속도(Veolcity)|(마우스 사용 가능)장치의 이동속도가 빨라지면, 붓의 크기가 줄어듭니다|
|방향(Direction)|(마우스 사용 가능)장치의 방향에 따라서 붓의 크기가 달라집니다. 방향에 따른 붓의 크기는 내장된 무작위 방식을 따릅니다.|
|기울기(Tilt)|붓의 크기가 스타일러스의 기울기에 따라 달라집니다|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|(마우스 사용 가능)붓의 크기가 `도구 옵션` → `크기` 슬라이더의 값과 0 사이의 무작위 값으로 바뀝니다|
|점차 사라짐(Fade)|작은 크기의 붓에서 시작해서 `도구 옵션` → `크기` 슬라이더의 붓의 크기로 끝납니다|

### 2-4. 각도(Angle)

|항목|설명|
|---|---|
|압력(Pressure)|설명 없음|
|속도(Veolcity)|설명 없음|
|방향(Direction)|설명 없음|
|기울기(Tilt)|설명 없음|

### 2-5. 색상(Color)
색은 기본값으로 도구상자의 전경색을 가져옵니다. 그러나, `페인트 동적 요소 편집기` 대화상자의 `색상(Color)`을 활성화시키면, 색을 활성화된 그라디언트에서 가져옵니다.

|항목|설명|
|---|---|
|압력(Pressure)|설명 없음|
|속도(Veolcity)|(마우스 사용 가능)장치의 움직임이 느리면, 색상을 활성화된 그라디언트의 오른쪽에서 가져옵니다. 장치의 움직임이 빠르면, 색상을 그라디언트의 왼쪽에서 가져옵니다.|
|방향(Direction)|(마우스 사용 가능)장치의 방향에 따라서 활성화된 그라디언트에서 색을 가져옵니다. 색을 선택하는 방식은 내장된 무작위 방식을 따릅니다.|
|기울기(Tilt)|설명 없음|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|색상을 활성화된 그라디언트에서 무작위로 가져옵니다.|
|점차 사라짐(Fade)|장치로 선을 긋는 동안 활성화된 그라디언트의 왼편의 색에서 시작해서 오른편의 색까지 순차적으로 가져옵니다. "점차 사라짐"의 동작은 도구 옵션 대화상자의 "페이드 옵션"에서 설정할 수 있습니다.|

### 2-6. 경도(Hardness)
`경도(Hardness)` 옵션은 흐릿한(fuzzy) 붓에만 적용할 수 있습니다.

|항목|설명|
|---|---|
|압력(Pressure)|설명 없음|
|속도(Veolcity)|(마우스 사용 가능)장치의 움직임이 느리면, 붓의 경계가 예리해집니다. 장치의 움직임이 빠르면, 붓의 경계가 흐려집니다.|
|방향(Direction)|설명 없음|
|기울기(Tilt)|설명 없음|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|(마우스 사용 가능)경계의 흐림이 무작위로 결정됩니다.|
|점차 사라짐(Fade)|(마우스 사용 가능)붓으로 스트로크를 그리면 경계가 예리했다가 흐려집니다. "점차 사라짐"의 동작은 도구 옵션 대화상자의 "페이드 옵션"에서 설정할 수 있습니다.|

### 2-8. 종횡비(Aspect Ratio)
도구 옵션 대화상자의 `종횡비(Aspect Ratio)` 슬라이더의 값이 기본값인 0.00이 아닌 다른 값이어야 동적 요소가 활성화됩니다. `종횡비(Aspect Ratio)` 슬라이더가 0보다 작으면 붓의 너비가 바뀌게 됩니다. `종횡비(Aspect Ratio)` 슬라이더가 0보다 크면 붓의 높이가 바뀌게 됩니다.

|항목|설명|
|---|---|
|압력(Pressure)|설명 없음|
|속도(Veolcity)|(마우스 사용 가능)붓의 `종횡비(Aspect Ratio)`가 장치의 움직임의 속도에 따라 달라집니다.|
|방향(Direction)|(마우스 사용 가능)붓의 `종횡비(Aspect Ratio)`가 붓의 방향에 따라 달라집니다. 값을 선택하는 방식은 내장된 무작위 방식을 따릅니다.|
|기울기(Tilt)|설명 없음|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|(마우스 사용 가능)붓의 `종횡비(Aspect Ratio)`가 무작위로 달라집니다.|
|점차 사라짐(Fade)|(마우스 사용 가능)`종횡비(Aspect Ratio)` 슬라이더의 값이 0보다 크면, 붓의 스트로크는 최대 높이에서 시작해서 `종횡비(Aspect Ratio)` 슬라이더의 값으로 끝납니다. `종횡비(Aspect Ratio)` 슬라이더의 값이 0보다 작으면, 붓의 스트로크는 최대 너비에서 시작해서 `종횡비(Aspect Ratio)` 슬라이더의 값으로 끝납니다. `점차 사라짐(Fade)`의 동작은 도구 옵션 대화상자의 `페이드 옵션`에서 설정됩니다.|

### 2-9. 간격(Spacing)
`간격(Spacing)`은 붓이 한번에 선을 그릴 때의 낱개의 붓 모양의 거리입니다. `간격(Spacing)` 옵션을 설정하면 스타일러스나 마우스를 사용할 때 영향을 미칩니다.

|항목|설명|
|---|---|
|압력(Pressure)|설명 없음|
|속도(Veolcity)|(마우스 사용 가능)장치의 움직임이 느리면, 붓의 간격이 넓어집니다. 장치의 움직임이 빠르면, 붓의 경계가 좁아집니다.|
|방향(Direction)|(마우스 사용 가능)붓의 `간격(Spacing)`이 붓의 방향에 따라 달라집니다. 값을 선택하는 방식은 내장된 무작위 방식을 따릅니다.|
|기울기(Tilt)|설명 없음|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|(마우스 사용 가능)붓의 `간격(Spacing)`이 무작위로 달라집니다.|
|점차 사라짐(Fade)|(마우스 사용 가능)붓의 `간격(Spacing)`이 넓게 시작해서 좁게 끝납니다. `점차 사라짐(Fade)`의 동작은 도구 옵션 대화상자의 `페이드 옵션`에서 설정됩니다.|

### 2-10. 동작 속도(Rate)
`동작 속도(Rate)` 옵션은 [에어브러시](./14-03-10-00-airbrush.md), [흐릿하게/선명하게](./14-03-15-00-blur-sharpen.md), [문지르기](./14-03-16-00-smudge.md) 도구에 적용됩니다. `동작 속도(Rate)` 옵션은 시간 기반으로 효과를 나타냅니다. 이 옵션이 동작하려면 전보다 빠르거나 더 느리게 움직여야 합니다.

`동작 속도(Rate)`의 양은 도구 옵션 대화상자의 `동작 속도(Rate)` 슬라이더의 값으로 설정할 수 있습니다.

### 2-11. 흐름(Flow)
`흐름(Flow)` 옵션은 [에어브러시](./14-03-10-00-airbrush.md)에서만 동작합니다. 더 많거나 적은 칠을 하게 됩니다. `흐름(Flow)`의 양은 도구 옵션 대화상자의 `흐름(Flow)` 슬라이더의 값으로 설정할 수 있습니다.

### 2-12. 지터(Jitter)
보통 붓으로 선을 그리면, 붓 자국이 매우 가깝게 붙어 표시됩니다.

`지터(Jitter)`를 추가하면, 북 자국이 선을 따라 흩어지게 됩니다. `지터(Jitter)`의 양은 도구 옵션 대화상자의 `지터(Jitter)` 옵션의 양의 슬라이더의 값으로 설정할 수 있습니다.

[관련 정보: 14.3.1.2.12. 지터 적용(Apply Jitter)](./14-03-01-02-12-apply_jitter.md)

|항목|설명|
|---|---|
|압력(Pressure)|압력을 더 세게 주면, 붓 자국의 흩어지는 정도가 줄어듭니다. 압력이 약해지면, 붓 자국의 흩어지는 정도가 도구 옵션 대화상자의 `지터(Jitter)` 옵션의 양까지 커집니다.|
|속도(Veolcity)|(마우스 사용 가능)장치의 움직임이 빠르면, 붓 자국의 흩어지는 정도가 줄어듭니다. 장치의 움직임이 느리면, 붓 자국의 흩어지는 정도가 도구 옵션 대화상자의 `지터(Jitter)` 옵션의 양까지 커집니다.|
|방향(Direction)|(마우스 사용 가능)붓 자국의 흩어지는 정도가 붓의 방향에 따라 달라집니다. 값을 선택하는 방식은 내장된 무작위 방식을 따릅니다.|
|기울기(Tilt)|설명 없음|
|휠/회전(Wheel)|설명 없음|
|무작위(Random)|(마우스 사용 가능)붓 자국의 흩어지는 정도가 무작위로 달라집니다.|
|점차 사라짐(Fade)|(마우스 사용 가능)붓 자국의 흩어지는 정도가 없는 것으로 시작해서 도구 옵션 대화상자의 `지터(Jitter)` 옵션의 양까지 커져서 끝납니다. `점차 사라짐(Fade)`의 동작은 도구 옵션 대화상자의 `페이드 옵션`에서 설정됩니다.|
